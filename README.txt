El proyecto sigue en desarrollo. Conectandose a la API pokeapi. Mando llamar el array de pokemones, su img, además de otras características cómo habilidades,
tipos, entre otras estadísticas. Es un buen ejemplo para relacionarse con las llamadas AJAX y el uso de fetch. En el proyecto se encuentran ambas herramientas.
En sí, sólamente se tiene que agregar el nombre del pokemon y presionar el botón "Busqueda", para limpiar los datos se usa el botón "Nueva búsqueda".
También se realizó un diseño en Figma para tomar de base en la maquetación.
Aún hay bastantes cosas que se pueden mejorar, desde diseño, UX, el mismo código que hace las request, jugar más con los objetos que tienen los JSON. Implementar más botónes que permiten actualizar los datos, así cómo otros elementos faltantes.
Popsteriormente se piensa terminar este proyecto implementandolo en una aplicacion web usando Spring Boot.
